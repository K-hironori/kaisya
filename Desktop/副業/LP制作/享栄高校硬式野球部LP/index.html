年会費と寄付はどう違うのですか？

+

年会費は、OB・OG会の組織的な運営と、現役選手への継続的な支援を目的とした年間の会費です。寄付は任意のご支援ですが、年会費は会員としての年間の会費として、安定した活動基盤を支えるものです。年会費をお納めいただくことで、会員特典もご利用いただけます。

毎年振り込む必要がありますか？

+

はい、年会費・応援会費ともに毎年お振り込みをお願いしております。毎年3月頃に事務局より納入のご案内をお送りいたしますので、4月末日までにお振り込みください。

領収書は発行されますか？

+

はい、ご希望の方には正式な領収書を発行いたします。お振り込み確認後、郵送にてお送りいたします。領収書が必要な方は、お振り込み時またはお振り込み後に事務局までお申し出ください。

住所や連絡先が変わった場合はどうすればいいですか？

+

ご住所やお電話番号、メールアドレスの変更は、事務局までご連絡ください。会報誌の郵送先や、各種ご案内が確実にお届けできるよう、変更があった際は速やかにお知らせいただけますようお願いいたします。

年会費は税務上の寄付金控除の対象になりますか？

+

申し訳ございませんが、年会費は寄付金控除の対象とはなりません。これは会員としての年会費であり、寄付とは性質が異なるためです。ただし、年会費以外に追加でご寄付をいただく場合は、別途ご相談ください。

振り込み名義人を変更したい場合はどうすればいいですか？

+

振り込み名義人を変更される場合は、事前に事務局までご連絡ください。ご家族名義での振り込みや、会社名義での振り込みも可能です。振り込み後に誰からの納入かを確認できるよう、事前にお知らせください。
